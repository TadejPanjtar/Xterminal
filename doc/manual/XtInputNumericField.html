Constructs a new XtInputNumericField.
Parameters:
__r - field bounds
__DefaultNumber - default number
__IntegerDigits - maximum integer digits allowed
__DecimalDigits - maximum decimal digits allowed
__Flags - options, which can be:

FIELD_READONLY - the content cannot be changed

FIELD_JUSTIFIED_RIGHT - the field is right-justified

ALLOW_NEGATIVE_VALUES - to allow entering negative values
