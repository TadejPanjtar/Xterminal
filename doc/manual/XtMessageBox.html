A message box
XtMessageBox(XRect __r, char *__Title, char *__Message, unsigned long __ButtonFlags)
Constructs a new XtMessageBox
Parameters:
__r - box bounds
__Title - box title
__Message - text to print
__ButtonFlags - buttons options, which can be:

YES_NO_BUTTON - "Yes"/"No" buttons

OK_CANCEL_BUTTON - "Ok"/"Cancel" buttons

OK_BUTTON - "Ok" button

CANCEL_BUTTON - "Cancel" button

virtual int Run()
"Runs" the dialog. Listens for events, process them, until the dialog is closed
